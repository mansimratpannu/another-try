# QR Attendance System

A fully functional QR Code Attendance System with login for teachers and students.

## Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Login with demo accounts

## Demo Accounts

| Role | Email | Password |
|------|-------|----------|
| Teacher | teacher@school.com | teacher123 |
| Student | john@school.com | student123 |

## Features

- **Teacher View**: Create sessions, generate QR codes, view attendance
- **Student View**: Scan QR codes, view attendance history
- **Professional Design**: Clean, modern UI

## Deploy to GitHub Pages

1. Go to Repository Settings → Pages
2. Select "main" branch as source
3. Save and wait for deployment

The site will be available at `https://yourusername.github.io/repository-name`
